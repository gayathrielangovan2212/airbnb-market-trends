# Airbnb Market Trends — NYC

Exploratory analysis of 25,000+ New York City Airbnb listings using R.

## Project Brief
As a consultant for a real estate start-up, I analysed NYC Airbnb data 
from multiple file formats to uncover short-term rental market insights.

## Key Findings
| Metric | Result |
|---|---|
| Earliest review | 2019-01-01 |
| Latest review | 2019-07-09 |
| Private room listings | 11,356 |
| Average nightly price | $141.78 |

## Tools & Packages
- R
- dplyr — data manipulation
- readr / readxl — importing CSV, TSV, Excel files
- stringr — text cleaning
- lubridate — date parsing
